# projects

project ごとの人間向けワークスペースを置くディレクトリです。

- `projects/<project-slug>/overview.md` を入口にする
- `projects/<project-slug>/knowledge/` に project 固有の知識メモを置く
- `projects/<project-slug>/meta/` に project 固有の辞書を置く
- `projects/<project-slug>/views/` に進行中の一覧 view を置く
- `projects/<project-slug>/closed-views/` に完了・中止済みの一覧 view を置く
- 追加・削除の具体手順は `../PROJECT_OPERATIONS.md` を参照する
